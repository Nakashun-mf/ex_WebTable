# WebTable Enhancer

Webサイト上のテーブルを**右クリック 1 発**で変換できる Chrome / Edge 拡張機能です。

## 主な機能

### 1. リッチ表示変換

ダサいテーブルをモダン UI に変換します。

- ストライプ行・ホバーハイライト・ダークヘッダー
- **列ソート**（数値・日付・テキストを自動判定、昇順→降順→リセット）
- **グローバル検索フィルター**（キーワード入力で行を絞り込み + 件数表示）
- **列フィルター**（列ごとのチェックボックスで値を絞り込み、Excel ライク）
- **列リサイズ**（ドラッグで列幅を自由に変更）
- **列並替**（ドラッグ&ドロップで列の順序を変更）
- **列の表示/非表示**（任意の列を一時的に非表示）
- **CSV エクスポート**（表示中の行・列を UTF-8 BOM 付き CSV でダウンロード）
- **行選択**（クリック・Shift・Ctrl でマルチ選択）
- **行コピー**（選択行をタブ区切りでクリップボードにコピー、または **Ctrl+C** でショートカットコピー）
- **ハイライト**（ダブルクリックまたは右クリックメニューで行を色付け）
- **列の表示/非表示管理**（非表示列がなくてもメニューから常時アクセス可能）

### 2. ツリー表示変換

「Level」「レベル」「階層」などの列があるテーブルを階層ツリーに変換します。

```
Level | 項目     | 金額
------+----------+------
  1   | 売上合計  | 1000   ← [−] をクリックで子を折りたたみ
  2   |   商品A  |  600
  3   |     明細1 |  300
  2   |   商品B  |  400
```

子ノードがある行に `−` / `+` ボタンが表れ、クリックで展開/折りたたみができます。

- **全展開 / 全折畳み**
- **レベル指定展開**（右クリックメニューから "レベル N まで展開"）
- **フィルター検索**（ツールバーの入力欄でキーワード絞り込み、祖先ノードも自動表示）
- **ハイライト**（ダブルクリックで行を色付け）

## 対応テーブル形式

| 形式 | 対応状況 |
|------|----------|
| `<thead>` + `<tbody>` がある標準的なテーブル | ✅ |
| `<tbody>` のみのテーブル（先頭行を自動でヘッダーに昇格） | ✅ |
| **全セル `<td>` のテーブル**（`<th>` がなくても動作） | ✅ |
| 複数 `<tbody>` を持つテーブル | ✅ |
| `<iframe>` 内のテーブル | ❌ |

## インストール

### 拡張機能として読み込む（開発者向け）

```bash
git clone https://github.com/nakashun-mf/ex_webtable.git
cd ex_webtable
npm install
npm run build
```

**Chrome の場合:**
1. `chrome://extensions` を開く
2. 右上「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」→ リポジトリのルートフォルダを選択

**Microsoft Edge の場合:**
1. `edge://extensions` を開く
2. 左下「開発者モード」を ON
3. 「展開して読み込む」→ リポジトリのルートフォルダを選択

## 使い方

1. 任意の Webサイトでテーブルの上を右クリック
2. ブラウザのコンテキストメニューから操作を選択

| メニュー | 効果 |
|--------|------|
| テーブルをリッチ表示に変換 | ソート・フィルター付きモダン UI |
| テーブルをツリー表示に変換 | +/− ボタンで階層展開 |
| テーブルを元に戻す | 変換前に復元 |

変換済みのテーブルを右クリックすると、さらに詳細な操作メニューが表示されます。

| 右クリックメニュー（変換済みテーブル） | 効果 |
|--------------------------------------|------|
| セルの内容をコピー | 右クリックしたセルのテキストをコピー |
| 選択したテキストをコピー | ドラッグ選択中のテキストをコピー |
| 選択した行をコピー / ヘッダーと選択した行をコピー | 選択行を TSV 形式でコピー |
| 非表示列の管理 | 列の表示/非表示を切り替え（常時表示）|

## 開発

```bash
npm run build   # src/ を content.js にバンドル
npm run watch   # ファイル変更を監視して自動ビルド
npm run lint    # ESLint でコードチェック
```

ソースコードは `src/` 以下に機能別モジュールとして分割されています。詳細は [docs/DESIGN.md](docs/DESIGN.md) を参照してください。
