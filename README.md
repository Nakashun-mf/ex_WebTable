# WebTable Enhancer

Webサイト上のテーブルを**右クリック 1 展**で変換できる Chrome 拡張機能です。

## 主な機能

### 1. リッチ表示変換
ダサいテーブルをモダン UI に変換します。

- ストライプ行・ホバーハイライト・ダークヘッダー
- **列ソート**（数値・日付・テキストを自動判定、昇順→降順→リセット）
- **グローバル検索フィルター**（キーワード入力で行を絞り込み + 件数表示）

### 2. ツリー表示変換

「Level」「レベル」「階層」などの列があるテーブルを階層ツリーに変換します。

```
Level | 項目     | 金額
------+----------+------
  1   | 売上合計  | 1000   ← [−] をクリックで期子を折りたたみ
  2   |   商品A  |  600
  3   |     明紶1 |  300
  2   |   商品B  |  400
```

子ノードがある行に `−` / `+` ボタンが表れ、クリックで展開/折りたたみができます。

## 対応テーブル形式

- `<thead>` + `<tbody>` がある標準的なテーブル
- `<tbody>` のみのテーブル（先頭行を自動でヘッダーに昼格）
- **全セル `<td>` のテーブル**（`<th>` がなくても動作）

## インストール

1. `git clone` でリポジトリをクローン
2. Chrome で `chrome://extensions` を開く
3. 右上「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」→ リポジトリのルートフォルダを選択

## 使い方

1. 任意の Webサイトでテーブルの上を右クリック
2. 表示されたメニューから操作を選択

| メニュー | 効果 |
|--------|------|
| テーブルをリッチ表示に変換 | ソート・フィルター付きモダン UI |
| テーブルをツリー表示に変換 | +/− ボタンで階層展開 |
| テーブルを元に戻す | 変換前に復元 |
