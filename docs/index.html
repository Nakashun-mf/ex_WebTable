テーブルを、
もっと使いやすく。

HTML テーブルを右クリック一発でリッチな表示に変換。検索・ソート・フィルター・ツリー表示など、本格的なデータ操作をそのままブラウザで。

GitHub で見る インストール方法

Rich View

リッチ表示

Excel ライクな操作をそのままブラウザで。あらゆる HTML テーブルに対応します。

🔍

グローバル検索

全列を横断してリアルタイムに絞り込み。マッチ件数もカウント表示。

⇅

列ソート

数値・日付・テキストを自動判定して昇順 / 降順に並べ替え。

☑

列フィルター

チェックボックス形式の Excel ライクフィルターを全列に装備。

↔

列リサイズ・並替

ドラッグで列幅を変更、ヘッダーを掴んで列順を自在に入れ替え。

👁

列の表示 / 非表示

不要な列をワンクリックで隠して、見たい情報だけに集中。

💾

CSV エクスポート

表示中のデータを即座に CSV で書き出し。フィルター結果にも対応。

✅

行選択・コピー

Shift / Ctrl クリックで複数行を選択してクリップボードへコピー。

🌟

行ハイライト

重要な行をハイライトしてマーキング。目視確認の効率が上がる。

Tree View

ツリー表示

階層データを持つテーブルをインタラクティブなツリーに自動変換します。

📋

階層の自動検出

Level 列・レベル列・アンダースコアインデントなど複数の形式に対応。

＋−

展開 / 折りたたみ

ノードごとに展開・折りたたみ。子孫ごとまとめての操作も可能。

📄

全展開・全折畳み

ワンクリックで全ノードを一括操作。全体像の把握に。

🏗

レベル指定展開

任意の深さまで展開して、中間レベルの構造を俯瞰できる。

Level列 レベル列 階層列 アンダースコアインデント

Developer

インストール（開発者向け）

Chrome ウェブストア未公開のため、開発者モードで読み込みます。

1
リポジトリをクローン git clone https://github.com/nakashun-mf/ex_webtable.git
2
依存関係インストール & ビルド npm install && npm run build
3
Chrome の拡張機能ページを開く アドレスバーに chrome://extensions と入力してアクセス
4
デベロッパーモードを ON にする 右上のトグルをオンにして「パッケージ化されていない拡張機能を読み込む」をクリック
5
フォルダを選択して完了 クローンしたディレクトリを選択すればインストール完了です
