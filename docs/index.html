WebTable Enhancer

テーブルを、もっと使いやすく。

Webサイト上の HTML テーブルを右クリック一発でリッチな表示に変換する Chrome 拡張機能です。

🔗 GitHub で見る プライバシーポリシー
✨ リッチ表示
🔍グローバル検索フィルター
⇅列ソート（数値・日付・テキスト）
☑列フィルター（Excel ライク）
↔列リサイズ・列並替
👁列の表示 / 非表示
💾CSV エクスポート
✅行選択・コピー（Shift / Ctrl）
🌟行ハイライト
🌳 ツリー表示
📃Level 列から階層を自動検出
➕➖展開 / 折りたたみ
📄全展開・全折畳み
🏗レベル指定展開
Level列 レベル列 階層列 アンダースコアインデント
🚀 インストール（開発者向け）
リポジトリをクローン：git clone https://github.com/nakashun-mf/ex_webtable.git
npm install && npm run build でビルド
Chrome で chrome://extensions を開く
「デベロッパーモード」を ON にして「パッケージ化されていない拡張機能を読み込む」
クローンしたフォルダを選択して完了
