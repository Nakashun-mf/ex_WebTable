# WebTable Enhancer — 設計書

## 概要

WebTable Enhancer は、任意の Webサイト上の HTML `<table>` を**右クリックメニュー**から変換できる Chrome 拡張機能です。
2 つの主要機能を提供します。

| 機能 | 右クリックメニュー | 内容 |
|--------|-----------------|------|
| リッチ表示 | テーブルをリッチ表示に変換 | モダン UI・ソート・グローバル検索 |
| ツリー表示 | テーブルをツリー表示に変換（レベル列） | 階層展開/折りたたみ |
| リセット | テーブルを元に戻す | 変換前に復元 |

## 機能詳細

### 機能 1 : リッチ表示変換

**変換前**
- ブラウザデフォルトのスタイルのみ
- ソート・フィルター機能なし

**変換後**
- モダン UI（ストライプ行・ホバーハイライト・ダークヘッダー）
- グローバル検索フィルター（キーワードで行を絞り込み + 件数表示）
- 列ソート（クリックで昇順→降順→リセット）

#### ソートロジック

| 優先度 | 判定方法 | 例 |
|---------|---------|----|
| 1 | 数値（カンマ・通貨記号除去） | `1,000` → 1000 |
| 2 | 日付（`Date.parse`） | `2024-01-15` |
| 3 | 日本語対応テキスト（`localeCompare('ja')`） | 「あいう」順 |

#### 全 `<td>` テーブルへの対応

`<th>` がなくすべて `<td>` でかかれたテーブルも対応しています。  
`ensureStructure()` が先頭行を `<thead>` に昇格させ、導出されたヘッダーセル（`<td>` でも可）にソート機能を付与します。

---

### 機能 2 : ツリー表示変換

**対象テーブルの条件**

- ヘッダー行に下記のいずれかの列名が存在すること  
  `Level` / `レベル` / `階層` / `Lv` / `Lv.` / `depth` / `深さ`（大小文字不問）
- その列の各データ行に整数値（1、 2、 3…）が入っている

**階層構造の解釈例**

```
| Level | 項目     | 金額 |
|-------|----------|------|
| 1     | 売上合計  | 1000 |  ← ルート（トグルボタン付き）
| 2     |   商品A  |  600 |  ← 売上合計の子
| 3     |     明紶1 |  300 |  ← 商品Aの子
| 3     |     明紶2 |  300 |  ← 商品Aの子
| 2     |   商品B  |  400 |  ← 売上合計の子
```

- `売上合計 (Lv1)` の 「−」 をクリック→ `商品A / 商品B` が非表示に
- `商品A (Lv2)` の 「−」 をクリック→ `明紶1 / 明紶2` が非表示に
- 再度クリックで 「+」 → 展開
- 子を持つノードが折りたたまれた状態で親を展開すると、子は非表示のまま保寝される

**アルゴリズム**

1. スタックを使った O(n) 親子関係構築
2. `cells[0]` にトグルボタンを挿入、葉ノードにはスペーサーを挿入（文字位置を揃える）
3. `cell.paddingLeft = 8 + (level − 1) × 20px` でインデント
4. 表示/非表示は `HTMLTableRowElement.hidden` で制御

## インストール方法

1. このリポジトリをクローン（または ZIP ダウンロード）
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」→ リポジトリのルートフォルダを選択

---

## 使い方

1. 任意の Webサイトを開く
2. テーブルの上で **右クリック**
3. メニューから目的の操作を選択

| メニュー項目 | 実行内容 |
|-----------|----------|
| テーブルをリッチ表示に変換 | ソート・フィルター付きモダン UI に変換 |
| テーブルをツリー表示に変換（レベル列） | Level列がある場合に階層ツリーに変換 |
| テーブルを元に戻す | 変換前の状態に復元 |

---

## 対応テーブルの形式

| 形式 | 対応状況 |
|------|----------|
| `<thead>` + `<tbody>` | ✅ そのまま使用 |
| `<tbody>` のみ（`<thead>` なし） | ✅ 先頭行を `<thead>` に自動昇格 |
| 全セル `<td>`（ヘッダーに `<th>` なし） | ✅ 先頭行を `<thead>` に自動昇格 |
| 複数 `<tbody>` | ✅ 全 `<tbody>` の行を列挙 |
| `<iframe>` 内のテーブル | ❌ 未対応 |

---

## 制限事項 / 既知の問題

- リッチ変換とツリー変換の同時適用は不可（一方を先に「元に戻す」する必要あり）
- JavaScript で動的生成されるテーブルは、生成後に右クリックする必要あり
- `iframe` 内のテーブルは非対応
